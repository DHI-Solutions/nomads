### Domain Services

1. Get the package by `yarn add @dhi-solutions/domain-services` in the root of the project.

2. Whenever you're ready, please `import { initialize } from '@dhi-solutions/domain-services'` at the top of the file.

3. Initialize the connector before rendering your application:

```javascript
await initialize({
  baseURL: 'DOMAIN_SERVICES_URL',
  localStorageKey: 'APPLICATION_NAME',
});
```

4. Hook into the instance of the API, when communication with Domain Services.
